(직원)시스템 관련 : (직원)로그인, (직원)로그아웃, (직원)언어선택, (직원) 공지하기, (직원)문의 사항 관리
(직원)메뉴 관련 : (직원)메뉴 주성, (직원)메뉴등록, (직원)메뉴 별 통계 조회

						... 추천 메뉴 정보(이름, 가격, 설명)

		+ 현금 결제하기 버튼(in : int 선택메뉴[ ]) > 현금 결제
			+ 현금 결제 수행(in : int 선택메뉴[ ]) ... 메뉴 정보(이름, 가격, 설명)
		+ 현금 결제 완료 메세지 출력(void)
		+ 주문 내역 화면 출력(void) > 주문 내역 화면
			+ 적립하기 버튼(void) > 적립금
				+ 개인정보 입력화면 출력(void)
				+ 적립하기 버튼(in : 전화번호, 생년월일) ... 적립금 정보(전화번호, 생년월일, 적립금)
		+ 신용카드 결제하기 버튼(in : int 선택메뉴[ ]) > 신용카드 결제
			+ 신용카드 결제 수행(in : int 선택메뉴[ ]) ... 메뉴 정보(이름, 가격, 설명)
		+ 신용카드 결제 완료 메세지 출력(void)
		+ 주문 내역 화면 출력(void) > 주문 내역 화면
			+ 적립하기 버튼(void) > 적립금
				+ 개인정보 입력화면 출력(void)
				+ 적립하기 버튼(in : 전화번호, 생년월일) ... 적립금 정보(전화번호, 생년월일, 주문번호, 적립금)
		+ 적립금 결제하기 버튼(in : int 선택메뉴[ ]) > 적립금 결제
			+ 적립금 결제 수행(in : int 선택메뉴[ ]) ... 메뉴 정보(이름, 가격, 설명)
